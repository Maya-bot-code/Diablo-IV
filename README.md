# Description

Ce projet est un site vitrine one-page entièrement réalisé en HTML et SCSS. Il a été développé dans le cadre de mon dernier projet SCSS à l'école pour valider mes compétences. Le site présente l’univers du jeu Diablo IV avec un design immersif et des effets interactifs.

# Fonctionnalités principales

- One-page scroll : navigation fluide sur une seule page.
- Animations CSS : effets de hover sur les boutons et images.
- Background dynamique : éléments visuels animés pour rendre le site vivant.

# Technologies utilisées

- HTML5 pour la structure du site.
- SCSS pour un style modulable et maintenable.
- CSS Animations pour les effets interactifs.

# Objectif pédagogique

- Maîtriser la mise en page avancée avec SCSS.
- Appliquer des animations CSS sur des éléments interactifs.
- Créer un site vitrine professionnel et esthétique.
